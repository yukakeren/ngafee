## 📱 Persyaratan Sistem

Sebelum melakukan instalasi, pastikan perangkat Anda memenuhi kriteria berikut:

* **Versi Android:** 5.0 (Lollipop) atau lebih tinggi.
* **Ruang Penyimpanan:** Minimal 50 MB tersedia.
* **Koneksi Internet:** Diperlukan untuk fitur peta dan navigasi secara real-time.

---

## 🛠 Langkah-langkah Instalasi

### 1. Download File APK
Unduh file **`ngafee.apk`** dari https://drive.google.com/file/d/1FiAZGl1N-UDNM5xFLfMumCAExoxMoB9c/view?usp=sharing dan simpan di memori internal perangkat Android Anda.

### 2. Aktifkan Instalasi dari Sumber Tidak Dikenal
Karena aplikasi ini diinstal di luar Google Play Store, Anda perlu memberikan izin khusus:

**Untuk Android 8.0 (Oreo) ke atas:**
1.  Buka **Settings** (Pengaturan).
2.  Pilih **Apps & notifications** (Aplikasi & notifikasi).
3.  Pilih **Advanced** (Lanjutan) → **Special app access** (Akses aplikasi khusus).
4.  Pilih **Install unknown apps** (Instal aplikasi tidak dikenal).
5.  Pilih browser atau file manager yang Anda gunakan untuk mengunduh APK.
6.  Aktifkan opsi **Allow from this source** (Izinkan dari sumber ini).

### 3. Instal Aplikasi
1.  Buka **File Manager** atau aplikasi pengelola file di perangkat Anda.
2.  Cari file `ngafee.apk` (biasanya berada di folder **Downloads**).
3.  Ketuk file APK tersebut.
4.  Ketuk tombol **Install** (Instal).
5.  Tunggu hingga proses instalasi selesai.
6.  Ketuk **Open** (Buka) untuk langsung menjalankan aplikasi, atau **Done** (Selesai).

### 4. Berikan Izin yang Diperlukan
Saat pertama kali membuka aplikasi, berikan izin berikut agar fitur berfungsi maksimal:
* **Lokasi:** Untuk menampilkan posisi Anda di peta dan panduan navigasi ke kafe terdekat.
* **Penyimpanan:** Untuk menyimpan data cache peta agar aplikasi lebih cepat (opsional).

> [!TIP]
> Pastikan Anda mengetuk **Allow (Izinkan)** saat muncul jendela permintaan izin agar aplikasi tidak mengalami kendala fitur.
